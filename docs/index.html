01 — Features

필요한 건 다 있습니다

정적 HTML/CSS/JS 작업에 필요한 실시간 미리보기 기능을 하나에.

🌐

실제 브라우저와 동일한 렌더링

로컬 서버 + iframe 구조라 fetch·ES 모듈·절대경로·웹폰트까지 그대로 동작합니다. 웹뷰 스킴의 제약이 없습니다.

<button class="cta">시작하기</button>
fetch('/api/data').then(...)
Live — 브라우저 그대로 동작
⚡

라이브 리로드

편집·저장 즉시 갱신. 링크된 CSS/JS를 고쳐도 자동 반영됩니다.

📝

저장 안 해도 반영

편집 중인 버퍼를 우선 제공해 타이핑하는 대로 미리보기.

📱

반응형 미리보기

Desktop / Tablet / Mobile 폭 전환.

🐞

친절한 에러 표시

쉬운 설명 + 원문을 유리 카드로.

⚙️

바로 조절하는 설정

툴바 톱니로 즉시 조절·자동 저장.

02 — Error overlay

에러가 나면, 화면이 알려줍니다

출력 패널을 안 열어도 됩니다. 원인을 사람 말로 설명하는 유리 카드가 떠오릅니다.

문제 2개✕
「doSomething」(이)라는 것을 찾을 수 없어요. 이름 오타이거나 스크립트를 안 불러왔을 수 있어요.
doSomething is not defined (index.html:14)
값이 비어있는 대상에서 「textContent」을(를) 읽으려 했어요.
Cannot read properties of null (reading 'textContent')

03 — Install

3단계면 끝

Marketplace에서 설치하거나 .vsix 파일로 바로 설치.

STEP 1

설치

확장 패널에서 HTML Live Viewer 검색 → Install

STEP 2

HTML 열기

아무 .html 파일을 열면 미리보기가 자동으로.

STEP 3

편집

코드를 수정하면 실시간으로 갱신됩니다.
